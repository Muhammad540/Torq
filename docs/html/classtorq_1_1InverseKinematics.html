Detailed Description

QP-based differential inverse kinematics solver.

Assembles a Quadratic Program from tasks (objective) and limits (constraints), then solves via OSQP. The solver is warm-started across ticks: the sparsity pattern is set once and only numerical values are updated on subsequent calls.

QP form:

\[ \min_{\Delta q}\;\tfrac{1}{2}\,\Delta q^\top P\,\Delta q + c^\top \Delta q \quad\text{s.t.}\quad G\,\Delta q \le h \]

Returns a velocity \(v = \Delta q / \Delta t\).
