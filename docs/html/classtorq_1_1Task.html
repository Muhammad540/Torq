Detailed Description

Abstract base class for kinematic tasks that contribute to the QP objective.

Each task defines an error \(e(q) \in \mathbb{R}^k\) and a Jacobian \(J(q) \in \mathbb{R}^{k \times n_v}\) such that the first-order task dynamics are:

\[ J(q)\,\Delta q = -\alpha\,e(q) \]

The base class converts this into a QP objective contribution:

\[ \min_{\Delta q}\;\tfrac{1}{2}\left\|W\bigl(J\,\Delta q + \alpha\,e\bigr)\right\|^2 \]

which expands to \(\tfrac{1}{2}\,\Delta q^\top H\,\Delta q + c^\top \Delta q\).

◆ computeQPObjective()

std::pair<Eigen::MatrixXd, Eigen::VectorXd> torq::Task::computeQPObjective ( const Configuration &  config) const

Compute this task's \((H, c)\) contribution to the QP objective.

Implements:

\[ \bar{J} = W\,J, \quad \bar{e} = W(-\alpha\,e) \]

\[ \mu^{\text{eff}} = \mu \cdot \bar{e}^\top \bar{e} \]

\[ H = \bar{J}^\top \bar{J} + \mu^{\text{eff}}\,I_{n_v}, \quad c = -\bar{J}^\top \bar{e} \]

Parameters
configCurrent robot configuration.
Returns
Pair (H, c) where H is \(n_v \times n_v\) and c is \(n_v \times 1\).
