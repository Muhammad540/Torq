Detailed Description

ST/STS/SMS series protocol on top of SCSPort — middle layer of the servo stack.

Same memory map and protocol as the SCServo SDK SMS_STS; the base is our SCSPort (POSIX or extensible to ESP32), so no Arduino dependency. Provides WritePosEx, SyncWritePosEx, ReadPos, Ping, EnableTorque, and sync read support. ServoDriver uses this for all bus I/O and radian conversion above.
