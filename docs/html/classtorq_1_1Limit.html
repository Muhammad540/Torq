Detailed Description

Abstract base class for kinematic limits (QP inequality constraints).

Each limit produces rows \((G, h)\) such that:

\[ G\,\Delta q \le h \]

These rows are stacked with those from other limits and passed to the OSQP solver as the constraint matrix.

Returns std::nullopt if there are no active constraints (e.g. no finite bounds in the model).
