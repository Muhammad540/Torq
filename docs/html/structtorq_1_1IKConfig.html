Detailed Description

Centralised store for all tunable IK parameters with sensible defaults.

Owned by the Controller; read-only access is available through RobotSystem::ikConfig(). Every field has a corresponding runtime setter on both Controller and RobotSystem.

See also
Parameter Tuning Guide for detailed descriptions and tuning recipes.
