Detailed Description

Regulate the SE(3) pose of a robot frame to a desired target.

The 6-D error is computed via the logarithmic map:

\[ e = \log_6\!\left(T_{\text{cur}}^{-1}\;T_{\text{des}}\right) = \begin{bmatrix} e_{\text{pos}} \\ e_{\text{ori}} \end{bmatrix} \in \mathbb{R}^6 \]

The 6-D cost vector is \([w_{\text{pos}}^{(3)},\;w_{\text{ori}}^{(3)}]\), allowing independent weighting of position and orientation tracking.

When no target is set the task is inert (contributes nothing to the QP).
