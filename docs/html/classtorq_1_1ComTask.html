Detailed Description

Regulate the center of mass position.

Error: \(e = \text{com}_{\text{actual}} - \text{com}_{\text{target}}\) (3D). Jacobian: pinocchio::jacobianCenterOfMass.

Critical for humanoid balance and quadruped stability. When no target is set the task is inert.
