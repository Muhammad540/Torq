Detailed Description

Abstract base class for Control Barrier Functions (CBFs).

A barrier defines a function \(h(q) \ge 0\) over the safe region of configuration space. The CBF condition:

\[ \frac{\partial h}{\partial q}\,\dot{q} + \alpha\,h(q) \ge 0 \]

is enforced as a QP inequality, and an optional safe-displacement cost regularises the objective toward a known-safe velocity.

See also
Barriers for mathematical details.
