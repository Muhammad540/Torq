Detailed Description

Top-level user-facing API that owns every subsystem.

RobotSystem is the single entry point for all robot operations: physics simulation, kinematics, IK-based control, and GUI interaction. It owns the HardwareInterface (simulation or real), KinematicsEngine, and Controller.

Typical usage:

torq::RobotConfig cfg;
cfg.scene_path = "scene.xml";
cfg.robot_model_path = "robot.urdf";
cfg.end_effector_frame = "ee_link";
torq::RobotSystem robot;
robot.initialize(cfg);
while (running) {
robot.update();
}
