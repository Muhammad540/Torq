Detailed Description

Snapshot of a robot configuration with pre-computed forward kinematics.

A Configuration holds a Pinocchio Model reference and a Data instance where FK has been computed for a given \(q\). It provides frame transforms, Jacobians, and manifold-aware integration.

Configurations are created by KinematicsEngine::makeConfiguration().
