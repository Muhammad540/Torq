Detailed Description

Minimum-distance barrier between two frames (spherical approximation).

\[ h(q) = \|p_1 - p_2\|^2 - d_{\min}^2 \]

The Jacobian uses position Jacobians in the world frame: \(\partial h / \partial q = 2(p_1 - p_2)^\top (J_1 - J_2)\).
