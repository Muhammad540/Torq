Detailed Description

Joint position (configuration) limits.

Constrains \(\Delta q\) so that \(q + \Delta q\) stays within \([q_{\min}, q_{\max}]\), with a proportional gain \(\gamma\) that smoothly decelerates the robot as it approaches a limit:

\[ \Delta q_j \le \gamma\,(q_j^{\max} - q_j), \quad -\Delta q_j \le \gamma\,(q_j - q_j^{\min}) \]

See also
Limits for a detailed discussion of the gain parameter.
