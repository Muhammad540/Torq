Detailed Description

Loads and owns the Pinocchio model; creates Configuration objects.

Supports loading from both URDF (.urdf) and MJCF (.xml) files. Joints listed in locked_joint_names are frozen via pinocchio::buildReducedModel so that IK only operates on the remaining actuated DOFs.
