Detailed Description

Joint acceleration limits with braking-distance bounds.

Two constraint types per joint:

  1. Finite-difference acceleration bound: \( -a_{\max}\,\Delta t^2 + \Delta q_{\text{prev}} \;\le\; \Delta q \;\le\; a_{\max}\,\Delta t^2 + \Delta q_{\text{prev}} \)
  2. Braking-distance bound (Flacco2015, DelPrete2018): \( |\dot{q}| \le \sqrt{2\,a_{\max}\,\text{dist\_to\_limit}} \)

The tighter of both constraints is enforced per joint.
