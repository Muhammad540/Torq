Detailed Description

ImGui + GLFW + OpenGL3 docking GUI for Torq.

Provides a 3D MuJoCo viewport, joint-control panel, Cartesian jog panel, and an IK tuning panel. The GUI does not own the RobotSystem — it only holds a non-owning pointer and reads/writes through the public API.
